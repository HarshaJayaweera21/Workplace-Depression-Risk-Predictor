

### **Dataset Details**



###### **Source and Link**



**Source:** Kaggle \[ https://www.kaggle.com/ ]

**Direct Link:** Depression Professional Dataset \[ https://www.kaggle.com/datasets/ikynahidwin/depression-professional-dataset/data ]



###### **Overview**



This tabular survey dataset provides realistic, non-medical data suitable for workplace mental health screening. It contains 2,054 records with 10 features (6 numerical, 5 categorical) and a binary target variable. No missing values or personal identifiers ensure ethical usability.



###### **Features**



Numerical Features :

* Age
* Work Pressure
* Job Satisfaction
* Work Hours
* Financial Stress



Categorical Features :

* Gender
* Sleep Duration
* Dietary Habits
* Suicidal Thoughts
* Family Mental Health





###### **Target Variable**



**Depression:** Binary classification (Yes = Depressed, No = Not Depressed)
